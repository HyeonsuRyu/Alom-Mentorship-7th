1. manifests : AndroidManifest.xml 파일이 들어있는 폴더이다. 이 파일에는 앱의 이름, 어떤 액티비티를 처음 시작할지 등 앱의 정보가 들어있다.
	안드로이드 어플리케이션을 구동하는데 필요한 설정값을 관리해주는 곳이다 

2. java : java 폴더에서는 자바 소스 코드를 입력할 수 있다. java 폴더에서 androidTest는 앱의 테스트를 위한 코드가 들어가는 부분이다.
	보이지 않는 모든 기능들을 담당.

3. res : resource와 관련된 내용이 들어갑니다. 이미지, 색깔, 폰트, 레이아웃 등이 담겨있는 폴더다. 겉으로 보이는 모든 것들을 담당.

4. Gradle Scripts : 프로젝트를 build 할 build script를 가지고 있는 파일.
	어플리케이션을 빌드하기위해 필요한 옵션,라이브러리 정보들이 들어있다.
