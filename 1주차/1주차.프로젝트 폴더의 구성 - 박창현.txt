1. manifests : AndroidManifest.xml 파일이 들어있는 폴더이다. 이 파일에는 앱의 이름, 어떤 액티비티를 처음 시작할지 등 앱의 정보가 들어있다.

2. java : java 폴더에서는 자바 소스 코드를 입력할 수 있다. java 폴더에서 androidTest는 앱의 테스트를 위한 코드가 들어가는 부분이다.

3. res : resource와 관련된 내용이 들어간다. 이미지, 색깔, 폰트, 레이아웃 등이 담겨있는 폴더이다.

4. Gradle Scripts : 프로젝트를 build 할 build script를 가지고 있는 파일이다.
