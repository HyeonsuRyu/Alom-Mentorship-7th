AndroidManifest -> 안드로이드 컨트롤 타워
		  안드로이드 어플을 구동하는데 필요한 설정값 관리

GradleScripts -> 어플 빌드 하기위한 옵션, 라이브러리 정보 저장

'둘 다 지금 당장 자세히 알 필요 없다'를 자세히 알게된 후 보았다.

res -> 리소스 폴더
         UI와 관련된 파일, 디자인, 문자열 리소스 저장
         앱에서 겉으로 보이는 모든 것 담당

java -> 클래스 관리
          앱 작동방식이 정의되어있다.

Activity_main.xml -> 앱 위에 실행될 화면

MainActivity.java -> Activity를 앱에 띄워준다

코드중 setContenView -> 표시될 화면 지정
